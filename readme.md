## Mengenai Lara54-Mix
Ini adalah repository untuk initial project Laravel 5.4 untuk bisa menggunakan Laravel-Mix di Windows dengan Homestead. Berikut adalah versi software penting yang digunakan:

- Homestead
- Windows 10 (Student Edition)
- Laravel 5.4
- Node v8.9.4
- NPM 5.6.0
